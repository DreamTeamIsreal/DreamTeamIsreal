## рЅБрѕЁрѕфрІФрЅх

* **рІеріЦрїЕ рѕЏрЅЁрѕерЅЦ:** рІюрїјрЅй ріЦрїЕрІјрЅйріЋ ріЦріЋрІ▓рІФрЅђрѕГрЅА ріЦріЊ ріЦріЋрІ▓рІ░рїЇрЇЅ рІФрѕхрЅйрѕІрѕЇрЇб
* **рІерЅАрІхріЋ рїЇріЋрЅБрЅ│:** рЅарїІрѕФ рїЇрЅдрЅй рІЎрѕфрІФ рІерЅхрЅЦрЅЦрѕГ рЅАрІхріќрЅй ріЦріЋрІ▓рЇѕрїарѕЕ рІФрѕўрЅ╗рЅ╗рѕЇрЇб
* **рІерІ│рЅ│ рѕЮрѕхрѕЇ рѕЏрѕ│рІФ:** рІерІюрїјрЅйріЋ рѕЮрѕГрїФрІјрЅй ріЦріЊ рІерЅ░рѕ│рЅхрЇј рІ│рЅ│ріЋ рЅарЅ░рѕўрѕѕріерЅ░ рїЇріЋрІЏрЅцрІјрЅйріЋ рІГрѕ░рїБрѕЇрЇб
* **рІеріЦрїЕрІјрЅй рѕЏрІЇрїФ:** рІерЅђрѕерЅА ріЦрїЕрІјрЅй ріЦріЊ рІеріЦріљрѕ▒ рѕўрїѕрѕѕрїФрІјрЅй ріарїарЅЃрѕІрІГ рІЮрѕГрІЮрѕГрЇб
* **рЅБрѕѕрЅЦрІЎ рЅІріЋрЅІ рІхрїІрЇЇ:** рѕѕрІЋрЅЦрѕФрІГрѕхрїЦрЇБ ріарѕерЅЦріЏрЇБ ріарѕЏрѕГріЏрЇБ рѕЕрѕ▓рІФріЏ ріЦріЊ ріЦріЋрїЇрѕірІЮріЏ рѕЎрѕЅ рІхрїІрЇЇрЇБ ріарѕхрЇѕрѕІрїі рѕ▓рѕєріЋ ріерЅђріЮ рІѕрІ░ рїЇрѕФ (RTL) ріарЅЁрїБрїФріЋ рїерѕЮрѕ«рЇб

## рѕўрїђрѕўрѕГ

рЇЋрѕ«рїђріГрЅ▒ріЋ рЅаріаріФрЅБрЅбрІФрІі рі«рѕЮрЇњрІЇрЅ░рѕГрІј рѕІрІГ рѕѕрѕЇрѕЏрЅхріЊ рѕѕрѕЎріерѕФ рІЊрѕІрѕЏрІјрЅй рѕѕрѕЏрѕхрігрІх рІерѕџріерЅ░рѕЅрЅхріЋ рѕўрѕўрѕфрІФрІјрЅй рІГріерЅ░рѕЅрЇб

### рЅЁрІхрѕў рѕЂріћрЅ│рІјрЅй

* Node.js (LTS рѕхрѕфрЅх рІГрѕўріерѕФрѕЇ)
* npm рІѕрІГрѕЮ Yarn (рІерїЦрЅЁрѕЇ ріарѕхрЅ░рІ│рІ│рѕф)

### рѕўрїФріЋ

1.  **рѕфрЇќрІџрЅ░рѕфрІЇріЋ ріГрѕјріЋ рІФрІхрѕГрїЅ:**
    ```bash
    git clone [https://github.com/DreamTeamIsrael/DreamTeamIsrael.git](https://github.com/DreamTeamIsrael/DreamTeamIsrael.git)
    cd DreamTeamIsrael
    ```

2.  **рїЦрїѕріърЅйріЋ рІГрїФріЉ:**
    рІѕрІ░ рЇЋрѕ«рїђріГрЅ▒ рѕхрѕГ рІГрѕѓрІ▒ ріЦріЊ ріарѕхрЇѕрѕІрїі рІерѕєріЉ рїЦрїѕріърЅйріЋ рІГрїФріЉ:
    ```bash
    npm install
    ```

## ріарЇЋрѕірігрѕйріЉріЋ рѕЏрѕхрігрІх

ріарЇЋрѕірігрѕйріЉріЋ рѕѕрѕЏрѕхрігрІх рІерѕџріерЅ░рѕЅрЅхріЋ рІ░рѕерїЃрІјрЅй рІГріерЅ░рѕЅ:

1.  **рІерѕЇрѕЏрЅх рѕ░рѕГрЅерѕГріЋ рѕЏрѕхрїђрѕўрѕГ:**
    ріарЇЋрѕірігрѕйріЉріЋ рЅарѕЇрѕЏрЅх рѕЂріљрЅ│ (ріерЇѕрїБріЋ рІ│рїЇрѕЮ рѕўрїФріЋ рїІрѕГ) рѕѕрѕЏрѕхрігрІхрЇБ рІГрѕЁріЋ рІГрїарЅђрѕЎ:
    ```bash
    npm run dev
    ```
    рІГрѕЁ рЅарЅ░рѕѕрѕЮрІХ рѕ░рѕГрЅерѕЕріЋ рЅа`http://localhost:5173` (рІѕрІГрѕЮ 5173 рѕхрѕФ рѕІрІГ ріерѕєріљ рѕїрѕІ рЇќрѕГрЅх) рІФрѕхрїђрѕЮрѕФрѕЇрЇб

2.  **рѕѕрѕЮрѕГрЅх рѕўрїѕріЋрЅБрЅх (ріарѕЏрѕФрїГ):**
    рІеріарЇЋрѕірігрѕйріЉріЋ рѕЮрѕГрЅх рІЮрїЇрїЂ рІерѕєріљ рѕхрѕфрЅх рѕѕрѕўрЇЇрїарѕГрЇБ рІГрѕЁріЋ рІФрѕѓрІ▒:
    ```bash
    npm run build
    ```
    рІГрѕЁ рЅа`dist` рІ│рІГрѕгріГрЅХрѕф рІЇрѕхрїЦ рІерЅ░рѕ╗рѕ╗рѕЅ рѕхрЅ│рЅ▓ріГ ріЋрЅЦрѕерЅХрЅйріЋ рІГрЇѕрїЦрѕФрѕЇрЇб

3.  **рІерѕЮрѕГрЅх рїЇріЋрЅБрЅ│ріЋ рѕЏрІерЅх (ріарѕЏрѕФрїГ):**
    рІерѕЮрѕГрЅх рїЇріЋрЅБрЅ│рІЇріЋ рЅаріаріФрЅБрЅбрІЇ рѕѕрѕЏрїѕрѕЇрїѕрѕЇ ріЦріЊ рѕѕрѕЏрІерЅхрЇБ рІГрѕЁріЋ рІГрїарЅђрѕЎ:
    ```bash
    npm run preview
    ```
    рІГрѕЁ ріерѕўрїФріЉ рЅарЇірЅх рІерЅ░рѕ░рѕФрІЇріЋ рѕхрѕфрЅх рѕѕрѕўрЇѕрЅ░рѕй рїарЅЃрѕџ ріљрІЇрЇб

## рїЦрЅЁрѕЮ рѕІрІГ рІерІІрѕЅ рЅ┤ріГріќрѕјрїѓрІјрЅй

* **рІерЇірЅх-рѕўрїерѕерѕ╗:** React, TypeScript, React Router, Tailwind CSS, `react-i18next`, Lucide React

## ріарѕхрЅ░рІІрїйрІќ рѕЏрІхрѕерїЇ

ріерѕЁрЅЦрѕерЅ░рѕ░рЅА рІерѕџрІ░рѕерїЅ ріарѕхрЅ░рІІрїйрІќрІјрЅйріЋ рЅарІ░рѕхрЅ│ ріЦріЋрЅђрЅарѕІрѕѕріЋ! рІГрѕЁріЋріЋ рЇЋрѕ«рїђріГрЅх рѕѕрѕЏрѕ╗рѕ╗рѕЇ рЇЇрѕІрїјрЅх ріФрѕјрЅхрЇБ ріЦрЅБріГрІј рІерѕџріерЅ░рѕЅрЅхріЋ рІ░рѕерїЃрІјрЅй рІГріерЅ░рѕЅ:

1.  рѕфрЇќрІџрЅ░рѕфрІЇріЋ Fork рІФрІхрѕГрїЅрЇб
2.  ріарІ▓рѕх рЅЁрѕГріЋрїФрЇЇ рІГрЇЇрїарѕЕ (`git checkout -b feature/your-feature-name`)рЇб
3.  рѕѕрІЇрїдрЅйріЋ рІФрІхрѕГрїЅ ріЦріЊ рЅарІ░ріЋрЅЦ рѕўрѕъріерѕФрЅИрІЇріЋ рІФрѕерїІрїЇрїАрЇб
4.  рѕѕрІЇрїдрЅйрІјріЋ Commit рІФрІхрѕГрїЅ (`git commit -m 'feat: Add new feature'`)рЇб
5.  рІѕрІ░ рЅЁрѕГріЋрїФрЇЅ Push рІФрІхрѕГрїЅ (`git push origin feature/your-feature-name`)рЇб
6.  рІерѕўрїјрЅ░рЅх рїЦрІФрЅё (Pull Request) рІГріГрЇѕрЅ▒рЇб

ріЦрЅБріГрІј рі«рІхрІј рІерЇЋрѕ«рїђріГрЅ▒ріЋ рІері«рІх рІ░рѕерїЃрІјрЅй рІерѕџрІФріерЅЦрѕГ рѕўрѕєріЉріЋ ріЦріЊ рЅ░рІЏрѕЏрїЁ рѕЎріерѕФрІјрЅйріЋ рІФріФрЅ░рЅ░ рѕўрѕєріЉріЋ рІФрѕерїІрїЇрїАрЇб
